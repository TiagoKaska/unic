Construa um JAR executável

Para rodar a aplicação pode ser importada no seu editor favorito como elicpse, importar as depedencias do maven e executar apartir da classe
UnicApplication.java como uma aplicação java normal.

Você pode construir o arquivo JAR com mvn package && java -jar target/unic-0.1.0.jar  
Então você pode executar o arquivo JAR:
java -jar target/unic-0.0.1-SNAPSHOT.jar
